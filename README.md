## Global Footwear Sales Dashboard

## About the Project

This project is an interactive sales dashboard built in Power BI to analyze global footwear sales performance. It transforms raw data into meaningful insights to support business decisions.
## What this Dashboard Shows

- Total Sales, Profit, and Units Sold
- Sales vs Profit by Brand
- Category performance distribution
- Country-wise sales map
- Customer and payment insights
## Tools Used

- Microsoft Power BI
- Power Query (Data Cleaning)
- DAX (Calculations & KPIs)
## Why this Project

This dashboard helps identify top-performing brands, profitable categories, and geographic trends. It demonstrates how Power BI can be used to solve real-world business problems.

- Data modeling
- Creating KPIs using DAX
- Designing interactive dashboards
